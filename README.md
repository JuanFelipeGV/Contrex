# CONTREX 🔐

Contrex es un sistema gestor de contraseñas
desarrollado en Java mediante Programación
Orientada a Objetos.

## Características

- Gestión de credenciales
- Generación de contraseñas seguras
- Cifrado básico
- Autenticación 2FA
- Interfaz gráfica con Java Swing

## Tecnologías usadas

- Java
- IntelliJ IDEA
- Swing
- GitHub
